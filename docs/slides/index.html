Professional Vedic astrology — birth chart, dasha predictions & personalized remedies, zero API cost

Pipeline
1
create
Ingest birth profile: name, DOB, time, place (3 predefined cities)
2
process
Calculate chart, 16 Vargas, Vimshottari Dasha (5 levels), Shadbala, Ashtakavarga, Yogas
3
predict
Compute hourly scores from current transit + Prana Dasha planet
4
remediations
Identify stressed planets, generate gemstone/mantra/color/puja/charity remedies
5
report
Render full natal report with career/finance/health/relationship/education predictions
Key Features
  • Birth chart (D1) + 16 varga divisional charts (D2–D60)
  • Vimshottari Dasha: 5-level hierarchy across 120-year cycle
  • Hourly predictions (30-day cache) with C/F/H/R/E ratings
  • Monthly + daily breakdown with best/worst hour windows
  • 8 remedy types: gemstone, mantra, color, puja, charity, dietary, hora/kaala, navagraha peeth
  • AI summaries via local Ollama LLM (no data leaves machine)
  • South Indian chart grid + ASCII renderer
  • Life events journal overlaid on dasha timeline
